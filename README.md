🚨 DisasterVision-AI

An AI-powered disaster response system that analyzes drone and satellite images to detect infrastructure damage and generate real-time actionable maps for emergency teams.

---

🌍 Problem

After natural disasters like earthquakes and floods, it is difficult to quickly assess damage across large areas. Manual analysis is slow and inefficient.

---

💡 Solution

This project uses Computer Vision (YOLOv8) to:

- Detect objects (people, vehicles, debris)
- Classify damage levels (Minor, Moderate, Severe)
- Generate a live map with markers
- Create a heatmap for high-risk zones

---

⚙️ Tech Stack

- Python
- YOLOv8 (Ultralytics)
- Google Colab
- Folium (Maps & Heatmaps)

---

🔥 Features

- 📸 Image-based damage detection
- 🧠 AI classification logic
- 🗺️ Interactive disaster map
- 🔥 Heat zone visualization
- ⚡ Fast and scalable pipeline

---

📊 Output

- 🔴 Severe damage areas
- 🟠 Moderate damage areas
- 🟢 Minor damage areas
- Heatmap showing critical zones

---

🚀 How to Run

1. Open the notebook in Google Colab
2. Upload disaster images
3. Run all cells
4. View generated map and heatmap
